Class Wallet​Cache​Service 

public class WalletCacheService

A service class used to store and retrieve Wallet objects such as LinearWallet and HDWallet from the devices disk. This class will use the secure enclave (keychain if not available) to generate a key used to encrypt the contents locally, and retrieve. The class can be used to take the decrypted JSON and convert it back into Wallet classes, ready to be used.
